# timer01

A simple timer application built primarily using **HTML**, **JavaScript**, and **CSS**.

## Overview

`timer01` is a lightweight, browser-based timer utility. Its main focus is speed, ease of use, and run-anywhere compatibility. Designed as a straightforward solution for basic timed tasks, this app is ideal for students, presenters, or anyone needing a quick timer.

## Features

- **Easy-to-use interface:** Just set your time and start!
- **Customizable duration:** Input any time you need.
- **Modern and responsive design:** Works well on mobile and desktop.
- **Core tech stack:**  
  - **HTML** (73.9%): Sets up the structure.
  - **JavaScript** (14.2%): Powers the timer logic and interactions.
  - **CSS** (11.9%): Styles the user interface.

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Stonywillis/timer01.git
cd timer01
```

### 2. Open in your browser

Simply open `index.html` in your preferred web browser. No installation or builds are needed.

## Usage

1. Set the desired timer duration.
2. Click on **Start** to begin the countdown.
3. Optionally, use **Pause** or **Reset** as needed.
